# Python / 数据科学

Python 是当今最流行的编程语言之一，在数据科学、人工智能、Web 开发等领域广泛应用。

## 为什么选择 Python？

- **语法简洁**：接近自然语言，学习曲线平缓
- **生态丰富**：拥有 NumPy、Pandas、Django 等大量优秀库
- **社区活跃**：全球最大的开发者社区之一
- **跨平台**：Windows、macOS、Linux 均可运行

## 学习路径

1. **Python3 基础** → 掌握语法与核心概念
2. **NumPy** → 科学计算基础
3. **Pandas** → 数据分析利器
4. **Matplotlib** → 数据可视化
5. **SciPy** → 高级科学计算
6. **Django / Flask** → Web 开发实战

## 快速开始

```python
# 你的第一个 Python 程序
print("Hello, Python!")

# 列表推导式
squares = [x**2 for x in range(10)]
print(squares)
```
