# Python3 教程

Python 是一种解释型、面向对象、动态数据类型的高级程序设计语言。Python3 是目前的主流版本。

## 简介

Python 由 Guido van Rossum 于 1989 年底发明，第一个公开发行版于 1991 年发布。

## 基础语法

```python
# 变量与数据类型
name = "Python"
version = 3.12
is_awesome = True

# 条件判断
if is_awesome:
    print(f"{name} {version} 太棒了！")
```

## 主要特点

- **易于学习**：语法简洁清晰
- **易于阅读**：强制缩进，代码风格统一
- **丰富的标准库**：内置模块覆盖网络、文件、数据库等
- **跨平台**：一次编写，到处运行
- **解释型语言**：无需编译，快速迭代
